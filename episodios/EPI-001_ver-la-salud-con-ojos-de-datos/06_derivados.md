# Derivados · EPI-001

## Empezar a mirar la salud con ojos de datos

**Episodio:** 1  
**Tipo de derivados:** Promocionales, profesionales y estratégicos  
**Estado:** Versión sintética aprobada  
**Criterio:** Máximo 5 productos, incluido el tráiler  

---

## 1. Criterio editorial

Este episodio no requiere una producción extensa de derivados.

Su función es abrir el podcast con una voz clara, una tesis fundacional y una sensibilidad profesional: aprender a mirar la salud con ojos de datos.

Por eso, los derivados deben cumplir tres condiciones:

1. Ayudar a descubrir el podcast.
2. Comunicar con claridad la tesis del episodio.
3. Ser producibles sin convertir el proyecto en una carga operativa.

No se producirán derivados académicos formales para este episodio.

No se producirán múltiples piezas repetidas para redes.

No se elaborarán documentos extensos adicionales salvo que aporten utilidad real.

# Producto 5 · Ficha práctica breve

## Objetivo

Ofrecer una pieza útil para docentes, coordinadores, auditores y gestores en salud.

## Título

**Trazabilidad formativa: seis preguntas para leer mejor un proceso educativo en salud**

## Contenido

Ante una clase, módulo, capacitación o inducción en salud, pregúntate:

1. ¿Qué competencia real se quiere garantizar?
2. ¿Qué mínimos técnicos, éticos y normativos no son negociables?
3. ¿Qué experiencias permiten formar esa competencia?
4. ¿Qué evidencia demuestra que la persona no solo escuchó, sino que sabe actuar?
5. ¿Qué datos deja el proceso educativo?
6. ¿Qué decisión toma la institución con esos datos?

## Cierre

Si los datos no modifican nada, el sistema registra, pero no aprende.
