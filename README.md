# Audiopodcast de periodismo académico sonoro

Repositorio personal de gestión editorial, documental y operativa del audiopodcast de Juan David Cortés Martínez.

Este proyecto es un audiopodcast de periodismo académico sonoro orientado a traducir problemas complejos de salud pública, enfermería, ciencia de datos, inteligencia artificial, instituciones, políticas públicas y sociedad en comprensión pública rigurosa, inteligible y narrativamente atractiva.

## Tesis de fondo

La realidad social y sanitaria contemporánea solo puede comprenderse adecuadamente cuando se leen conjuntamente sus dimensiones humanas, institucionales, políticas, tecnológicas y analíticas.

## Propósito del repositorio

Este repositorio organiza el archivo interno del podcast. Su función es conservar la trazabilidad editorial, estructurar cada episodio, documentar decisiones, preparar productos derivados y facilitar la publicación profesional en plataformas de podcast como Captivate.fm.
